Assignment:

Create a website and sketch up an idea for a possible final project.

Make a website! You will use this website to document your work for the rest of the semester. You are welcome to use any tools you'd like to accomplish this, but if you don't have any previous web development experience, you'll likely want to stick with the Jekyll template we covered in class. as well as the HTML and CSS techniques that we learned in class -- however, you will likely need to spend some time researching new techniques in order to customize your site. The goal for this week is to make the site functional, i.e., that you're able to use it to host your assignments in a way that others are able to find them. You can (and should) continue refining the design when you have free time over the semester. It's fine to look for Bootstrap templates as references, but you must write all code from scratch. The assignment is due by this Thursday.

Your website must at least include all of the following features:

  • A header section in the form of a multiline comment, specifying the class name, your name, your email, assignment number and your choice of title
  • An index.html page that contains links to weekly (assignment) pages
  • Stand-in or "dummy" content for the weekly pages, which you will replace with your actual assignments as you go. It's not necessary to make a stand-in for each week yet, but the intention should be clear
  • An "About" page, including a paragraph about your hobbies, interests, and why you're taking this course, as well as a photograph so we can more quickly learn each others' names
  • At least 3 different classes with attributes defined in a separate CSS file
