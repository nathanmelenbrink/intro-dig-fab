Programming with Arduino

for() Loops

The for() structure performs repetitive calculations and is structured like this:


for (init; test; update) {
  statements 
}

Let's say you want to incrementally brighten an LED. You might do that with repetitive calculations:


// Dim an LED using a PWM pin
int PWMpin = 10;  // LED in series with 470 ohm resistor on pin 10

void setup() {
  // no setup needed
}

void loop() {
    analogWrite(PWMpin, 0);
    delay(100);
    analogWrite(PWMpin, 50);
    delay(100);
    analogWrite(PWMpin, 100);
    delay(100);
    analogWrite(PWMpin, 150);
    delay(100);
    analogWrite(PWMpin, 200);
    delay(100);
    analogWrite(PWMpin, 250);
    delay(100);
  }
}

However, the same result can be generated with fewer lines of code using a for loop:


// Dim an LED using a PWM pin
int PWMpin = 10;  // LED in series with 470 ohm resistor on pin 10

void setup() {
  // no setup needed
}

void loop() {
  for (int i = 0; i <= 255; i=i+50) {
    analogWrite(PWMpin, i);
    delay(100);
  }
}

This flexible pattern allows you to easily change the resolution of the fade:


// Dim an LED using a PWM pin
int PWMpin = 10;  // LED in series with 470 ohm resistor on pin 10

void setup() {
  // no setup needed
}

void loop() {
  for (int i = 0; i <= 255; i++) {
    analogWrite(PWMpin, i);
    delay(10);
  }
}
